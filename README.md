# Calculatore

![GitHub all releases](https://img.shields.io/github/downloads/ArjunRadhakrishnanv/Calculatore/total)

A streamlined version of the original Fossify Calculator, designed for speed, simplicity, and minimal footprint.

**Version:** 1.0.1

**Key Features:**

*   **User Interface:**
    *   **Circular Buttons:** Modern circular button design with custom ripple animations.
    *   **Dark Theme:** A permanent high-contrast black theme (including Status Bar and Navigation Bar) for visual clarity.
    *   **Portrait Mode:** Screen orientation is set to portrait for consistent usability.
    *   **Parenthesis Support:** Dedicated buttons for `(` and `)` are integrated into the main keypad for advanced calculations.
*   **Date Calculator:**
    *   **Calculate Difference:** Easily calculate the duration (years, months, days) between two specific dates.
    *   **Add/Subtract Time:** Add or subtract days, months, or years from a starting date to find a future or past date.
    *   **Access:** Accessible via the calendar icon on the top-left of the main screen.
*   **Performance:**
    *   **Optimized Animations:** Fast button press and transition animations (50ms).
    *   **Reduced Size:** Unused resources, widgets, and customization engines have been removed. The application is localized to English only.
*   **Privacy:** Fully functional offline. No internet permission required.

**Original Project:**
Based on [Fossify Calculator](https://github.com/FossifyOrg/Calculator).
